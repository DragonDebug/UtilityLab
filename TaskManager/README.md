# TaskManager

A browser-based task manager for tracking tasks, items, statuses, and reporting in one place.

**Status:** In Development

## How To Use

1. Open `index.html` in your web browser.
2. Add tasks, update task details, and use the built-in views and reports.
3. Keep using the same browser on the same machine if you want to keep the saved data available.

## Note

This app runs as a local static page and stores its data in the browser on the device you use.
