# TaskManagerModular

The modular version of TaskManager. This folder is being reorganized into smaller files while keeping the same core app behavior.

**Status:** Under Development

## How To Use

1. Open `index.html` in your web browser.
2. Use it like the main TaskManager app while the project continues moving code into smaller modules.

## Note

This folder is mainly for the ongoing refactor of TaskManager, so it may change more often than the main app.
