# GetFileNamesToTextFile

Exports file names from a folder into a text file for quick reference.

**Status:** Active

## How To Use

1. Run `GetFileNamesToTextFile.ps1` or `GetFileNamesToTextFile.bat`.
2. If needed, pass `-FolderPath` to scan a different folder.
3. The script saves the results as a text file after sorting the file names.

## Note

The output path is currently hardcoded inside the script, so update that path before using this tool on a different machine or project folder.
