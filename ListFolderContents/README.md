# ListFolderContents

Creates a text list of files and folders using saved settings for filtering and output.

**Status:** Active

## How To Use

1. Open `Settings\List-FolderContents-Settings.ps1` or `Settings\List-FolderContents-Settings.bat` to choose your filters and output file name.
2. Run `List-FolderContents.ps1` or `List-FolderContents.bat`.
3. The tool writes the result to the output file defined in the saved settings.

## Note

This version reads its settings from the `Settings` folder in this project and scans the folder where the script is placed.
